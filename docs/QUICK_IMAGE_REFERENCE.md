## 🔧 TROUBLESHOOTING

**Images not showing?**
1. Check filename matches EXACTLY (case-sensitive)
2. Verify file is in correct folder
3. Hard refresh browser (Ctrl+F5)
4. Check browser console for errors (F12)

**Images too large/small?**
1. Resize to exact dimensions specified
2. Use tools like: Photoshop, GIMP, or online resizers
3. Maintain aspect ratio

**File size too big?**
1. Reduce JPEG quality to 80%
2. Use WebP format
3. Use image compression tools like TinyPNG
