## 📸 Image Upload Guide

### **Required Images for New Packages:**

Each new tour needs these images in its folder:

1. **hero.jpg** (Required) - Main tour image (1920x1080px recommended)
2. **gallery-1.jpg** - Wildlife/landscape shot
3. **gallery-2.jpg** - Accommodation photo
4. **gallery-3.jpg** - Activity/experience shot
5. **gallery-4.jpg** - Additional highlight

**Image Specifications:**
- Format: JPG or WebP
- Size: Under 500KB each (optimized)
- Dimensions: 1920x1080px (16:9 ratio)
- Quality: High but compressed for web

**Upload Method:**
Place images in respective folders under `public/images/tours/[tour-slug]/`
