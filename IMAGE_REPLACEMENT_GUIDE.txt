# HOW TO REPLACE PLACEHOLDERS WITH REAL IMAGES
# ============================================================================
#
# STEP 1: Generate Placeholders
# ------------------------------
# Open this file in browser: senzalucesafaris\generate-placeholders.html
# Click "Generate All Placeholders"
# Download all images to: senzalucesafaris\public\images\placeholders\
#
# STEP 2: Replace with Real Images
# ---------------------------------
# For each placeholder image:
# 1. Find the real image you want to use
# 2. Resize it to the dimensions shown above (800x600 or 1920x1080)
# 3. Save it with the EXACT SAME filename
# 4. Place it in: senzalucesafaris\public\images\placeholders\
# 5. The website will automatically show the new image
#
# STEP 3: Verify
# --------------
# Refresh your browser (Ctrl+F5 for hard refresh)
# Check that all images display correctly
#
# ============================================================================
# RECOMMENDED IMAGE SPECIFICATIONS
# ============================================================================
#
# Hero Images (page backgrounds):
# - Size: 1920x1080 pixels or 1920x600 pixels
# - Format: JPEG
# - Quality: 80-90%
# - File size: Under 500KB
#
# Content Images (cards, sections):
# - Size: 800x600 pixels
# - Format: JPEG or WebP
# - Quality: 80-85%
# - File size: Under 300KB
#
# General Tips:
# - Use landscape orientation (wider than tall)
# - Keep file sizes small for faster loading
# - Use descriptive, relevant images
# - Ensure good lighting and composition
# - Consider mobile viewing (images will be cropped on small screens)
#
